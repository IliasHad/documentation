title: "Limits"
---

## Cloning

The Builder only support Github repositories.

## Timing

**A build can take up to 30 minutes**. After 30 minutes, the build timeout and
result in a failed build.

<Tip>
  Need more than 30 minutes to execute long tasks? Reach out to us on
  [Discord](https://discord.gg/defer)
</Tip>

## Background functions

- The Builder only support having one `defer` folder per project.
- Background functions must be named. They can not be anonymous.
- Background function names must be unique

## Cache

The Builder does not support npm cache
