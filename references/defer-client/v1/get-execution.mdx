---
title: "getExecution()"
icon: function
---

The `getExecution()` function retrieves an execution by its ID and may include
its result based on the execution state.

<CodeGroup>

```ts Get Execution
import { getExecution } from "@defer/client";
import { sayHello } from "./defer/sayHello";

const { id } = await sayHello("John Doe");
const response = await getExecution(id);

console.log(response.state, response.result);
```

```ts defer/sayHello.ts
import { defer } from "@defer/client";

const sayHello = async (name: string) => console.log(`hello ${name}!`);

export default defer(sayHello);
```

</CodeGroup>
