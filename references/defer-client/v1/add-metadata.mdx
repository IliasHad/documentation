<Info>
  **Deprecated**

Prefer [`assignOptions()`](/references/defer-client/v1/assign-options) as
`addMetadata()` will be dropped in `2.0`.

</Info>

The function `addMetadata()` adds the `metadata` attribute to a deferred
function. By default, no metadata is added to the execution.

Example:

<CodeGroup>

```ts index.ts
import { addMetadata } from "@defer/client";
import { sayHello } from "./defer/sayHello";

const delayedSayHello = addMetadata(sayHello, { userId: "1", foo: "bar" });
await delayedSayHello("John Doe");
```

```ts defer/sayHello.ts
import { defer } from "@defer/client";

const sayHello = async (name: string) => console.log(`hello ${name}!`);

export default defer(sayHello);
```

</CodeGroup>
